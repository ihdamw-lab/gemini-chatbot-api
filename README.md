Website ini merupakan tugas akhir dari sesi pembelajaran, yang menggabungkan berbagai tugas dan materi dari sesi-sesi sebelumnya. 

Salah satu fitur utama website ini adalah chatbot AI cerdas yang terintegrasi. Chatbot ini memungkinkan pengguna untuk berinteraksi dengan cara yang lebih dinamis dan multifungsi. Pengguna dapat mengirimkan berbagai jenis file, seperti gambar, dokumen, atau audio, untuk mendapatkan respons yang relevan dari AI.

Fitur chatbot meliputi:
- Pengguna bisa mengirim gambar, dokumen, atau file audio.
- Pengguna dapat menambahkan instruksi khusus (prompt) saat mengirim file, atau jika tidak ada prompt, sistem akan menggunakan instruksi default:
  - Untuk gambar: "Tolong jelaskan gambar ini secara detail."
  - Untuk dokumen: "Tolong buatkan ringkasan singkat dari dokumen ini."
  - Untuk audio: "Tolong buatkan transkrip dan ringkas isi audio ini." 

Chatbot ini dirancang untuk memberikan pengalaman yang mudah dan intuitif, membantu pengguna dalam berbagai kebutuhan, mulai dari deskripsi visual hingga ringkasan informasi. Website ini menunjukkan integrasi teknologi AI modern dengan aplikasi praktis sehari-hari.
